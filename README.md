# FC TamiLionS Website

A professional football academy profile website built with React, Vite, and Tailwind CSS.

## Setup

1. `npm install`
2. `npm run dev`

## Structure

- `src/components`: Reusable UI components.
- `src/pages`: Individual route pages.
- `src/data`: `teamData.json` acts as the CMS.

## Deployment

Run `npm run build` to generate the `dist` folder, which can be deployed to Netlify, Vercel, or GitHub Pages.
